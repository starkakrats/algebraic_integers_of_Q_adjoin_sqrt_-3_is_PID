    def Set.sigma {ι : Type u_1} {α : ι → Type u_3} (s : Set ι) (t : (i : ι) → Set (α i)) :
    Set ((i : ι) × α i)

    Indexed sum of sets. s.sigma t is the set of dependent pairs ⟨i, a⟩ such that i ∈ s and a ∈ t i.

    Equations
    • Set.sigma s t = {x : (i : ι) × α i | x.fst ∈ s ∧ x.snd ∈ t x.fst}
