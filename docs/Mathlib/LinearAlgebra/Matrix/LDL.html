    LDL decomposition #

    This file proves the LDL-decomposition of matrices: Any positive definite matrix S can be decomposed as S = LDLр┤┤ where L is a lower-triangular matrix and D is a diagonal matrix.

    Main definitions #

    • LDL.lower is the lower triangular matrix L.
    • LDL.lowerInv is the inverse of the lower triangular matrix L.
    • LDL.diag is the diagonal matrix D.

    Main result #

    • LDL.lower_conj_diag states that any positive definite matrix can be decomposed as LDLр┤┤.
