              X ⟶ X'
  ↘      ↘
    S ⟶ S'
  ↗      ↗
Y ⟶ Y'

              This iso witnesses the fact that

                X ×[S] Y ⟶ (X' ×[S'] Y') ×[Y'] Y
      |                  |
      |                  |
      ↓                  ↓

              (X' ×[S'] Y') ×[X'] X ⟶ X' ×[S'] Y'

              is a pullback square. The diagonal map of this square is pullback.map. Also see pullback_lift_map_is_pullback.
