    Type u_2

    A positive cone in a ring consists of a positive cone in underlying AddCommGroup, which contains 1 and such that the positive elements are closed under multiplication.

    • nonneg : α → Prop
    • pos : α → Prop
    • pos_iff : ∀ (a : α), AddCommGroup.PositiveCone.pos self.toPositiveCone a ↔ AddCommGroup.PositiveCone.nonneg self.toPositiveCone a ∧ ¬AddCommGroup.PositiveCone.nonneg self.toPositiveCone (-a)
    • zero_nonneg : AddCommGroup.PositiveCone.nonneg self.toPositiveCone 0
    • add_nonneg : ∀ {a b : α}, AddCommGroup.PositiveCone.nonneg self.toPositiveCone a → AddCommGroup.PositiveCone.nonneg self.toPositiveCone b → AddCommGroup.PositiveCone.nonneg self.toPositiveCone (a + b)
    • nonneg_antisymm : ∀ {a : α}, AddCommGroup.PositiveCone.nonneg self.toPositiveCone a → AddCommGroup.PositiveCone.nonneg self.toPositiveCone (-a) → a = 0
    • one_nonneg : AddCommGroup.PositiveCone.nonneg self.toPositiveCone 1

      In a positive cone, 1 is nonneg

    • mul_pos : ∀ (a b : α), AddCommGroup.PositiveCone.pos self.toPositiveCone a → AddCommGroup.PositiveCone.pos self.toPositiveCone b → AddCommGroup.PositiveCone.pos self.toPositiveCone (a * b)

      In a positive cone, if a and b are pos then so is a * b
