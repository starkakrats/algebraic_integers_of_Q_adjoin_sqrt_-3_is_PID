    Omega Complete Partial Orders #

    An omega-complete partial order is a partial order with a supremum operation on increasing sequences indexed by natural numbers (which we call œâSup). In this sense, it is strictly weaker than join complete semi-lattices as only œâ-sized totally ordered sets have a supremum.

    The concept of an omega-complete partial order (œâCPO) is useful for the formalization of the semantics of programming languages. Its notion of supremum helps define the meaning of recursive procedures.
