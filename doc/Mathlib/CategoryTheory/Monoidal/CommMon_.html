    A commutative monoid object internal to a monoidal category.

    • one : 𝟙_ C ⟶ self.X

    • mul_assoc : CategoryTheory.CategoryStruct.comp (CategoryTheory.MonoidalCategory.tensorHom self.mul (CategoryTheory.CategoryStruct.id self.X)) self.mul = CategoryTheory.CategoryStruct.comp (CategoryTheory.MonoidalCategory.associator self.X self.X self.X).hom (CategoryTheory.CategoryStruct.comp (CategoryTheory.MonoidalCategory.tensorHom (CategoryTheory.CategoryStruct.id self.X) self.mul) self.mul)
