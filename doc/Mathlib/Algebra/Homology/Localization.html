    The category of homological complexes up to quasi-isomorphisms

    Given a category C with homology and any complex shape c, we define the category HomologicalComplexUpToQis C c which is the localized category of HomologicalComplex C c with respect to quasi-isomorphisms. When C is abelian, this will be the derived category of C in the particular case of the complex shape ComplexShape.up ℤ.

    Under suitable assumptions on c (e.g. chain or cochain complexes), we shall show that HomologicalComplexUpToQis C c is also the localized category of HomotopyCategory C c with respect to the class of quasi-isomorphisms (TODO).
