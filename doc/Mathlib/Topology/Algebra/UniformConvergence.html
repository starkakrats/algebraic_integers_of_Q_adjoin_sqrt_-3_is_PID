    Algebraic facts about the topology of uniform convergence #

    This file contains algebraic compatibility results about the uniform structure of uniform convergence / ùîñ-convergence. They will mostly be useful for defining strong topologies on the space of continuous linear maps between two topological vector spaces.

    Main statements #

    • UniformFun.uniform_group : if G is a uniform group, then Œ± ‚Üí·µ§ G a uniform group
    • UniformOnFun.uniform_group : if G is a uniform group, then for any ùîñ : Set (Set Œ±), Œ± ‚Üí·µ§[ùîñ] G a uniform group.
    • UniformOnFun.continuousSMul_induced_of_image_bounded : let E be a TVS, ùîñ : Set (Set Œ±) and H a submodule of Œ± ‚Üí·µ§[ùîñ] E. If the image of any S ‚àà ùîñ by any u ‚àà H is bounded (in the sense of Bornology.IsVonNBounded), then H, equipped with the topology induced from Œ± ‚Üí·µ§[ùîñ] E, is a TVS.

    Implementation notes #

    Like in Topology/UniformSpace/UniformConvergenceTopology, we use the type aliases UniformFun (denoted Œ± ‚Üí·µ§ Œ≤) and UniformOnFun (denoted Œ± ‚Üí·µ§[ùîñ] Œ≤) for functions from Œ± to Œ≤ endowed with the structures of uniform convergence and ùîñ-convergence.
