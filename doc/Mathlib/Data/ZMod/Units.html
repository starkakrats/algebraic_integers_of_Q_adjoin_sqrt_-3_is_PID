    def ZMod.unitsMap {n : ℕ} {m : ℕ} (hm : n ∣ m) :
    (ZMod m)ˣ →* (ZMod n)ˣ

    unitsMap is a group homomorphism that maps units of ZMod m to units of ZMod n when n divides m.
