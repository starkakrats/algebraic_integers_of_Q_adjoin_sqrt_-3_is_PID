    The Ahlswede-Zhang identity #

    This file proves the Ahlswede-Zhang identity, which is a nontrivial relation between the size of the "truncated unions" of a set family. It sharpens the Lubell-Yamamoto-Meshalkin inequality Finset.sum_card_slice_div_choose_le_one, by making explicit the correction term.

    For a set family ùíú over a ground set of size n, the Ahlswede-Zhang identity states that the sum of |‚ãÇ B ‚àà ùíú, B ‚äÜ A, B|/(|A| * n.choose |A|) over all set A is exactly 1. This implies the LYM inequality since for an antichain ùíú and every A ‚àà ùíú we have |‚ãÇ B ‚àà ùíú, B ‚äÜ A, B|/(|A| * n.choose |A|) = 1 / n.choose |A|.
