    Type (max u_4 u_5)

    An isometry between two quadratic spaces M₁, Q₁ and M₂, Q₂ over a ring R, is a linear map between M₁ and M₂ that commutes with the quadratic forms.

    • toFun : M₁ → M₂
    • map_add' : ∀ (x y : M₁), AddHom.toFun self.toAddHom (x + y) = AddHom.toFun self.toAddHom x + AddHom.toFun self.toAddHom y
    • map_smul' : ∀ (r : R) (x : M₁), AddHom.toFun self.toAddHom (r • x) = (RingHom.id R) r • AddHom.toFun self.toAddHom x
    • map_app' : ∀ (m : M₁), Q₂ (AddHom.toFun self.toAddHom m) = Q₁ m

      The quadratic form agrees across the map.
